# auth.md — mcp-scorecard

**Agent audience:** any AI agent or developer. **No authentication is required — mcp-scorecard is fully public and runs locally.**

- **Tool:** runs on your machine via `npx -y mcp-scorecard <target>` (stdio, no API key, no account).
- **Site/data:** `https://mcp-scorecard.vercel.app/leaderboard.json` is openly readable (CORS `*`).
- **License:** MIT (code) — free to use, audit, and redistribute.

There are no protected resources, so there is no OAuth authorization server or registration flow to discover. Use everything immediately and anonymously.

Source: https://github.com/davidmosiah/mcp-scorecard
